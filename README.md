# Apex Data – Analyse de sessions sur circuit

## Description
Apex Data est une application web front-end destinée aux pilotes de sport automobile et de moto. Elle permet d’enregistrer, analyser et comparer les données de sessions sur circuit telles que les temps au tour, les réglages, les conditions météo et l’évolution des performances.

## Fonctionnalités
- Enregistrement de sessions avec des données détaillées (circuit, date, moto, pneus, météo, temps au tour)
- Tableau de bord affichant un résumé des performances
- Comparaison de sessions afin d’analyser la progression dans le temps
- Stockage local des données dans le navigateur

## Technologies utilisées
- HTML
- CSS
- JavaScript

## Utilisation
1. Télécharger le projet ou cloner le dépôt
2. Ouvrir le fichier `index.html` dans un navigateur web
3. Commencer à enregistrer et analyser les sessions de roulage
